# Repertoire

A Raycast AI-powered classical music manager for organizing, searching, and managing your classical music collection.

## Features

- Browse and search your classical music collection
- AI-powered metadata enrichment using Raycast AI
- Integration with Discogs for covers and catalog information
- Organize by composer, work, performer, and recording

## Development

This is a Raycast extension. To develop:

```bash
npm install
npm run dev
```
